Quick Overview

FileSense isn't just a sorter; it's a taxonomist. If a file fits an existing folder, it moves it there. If it encounters a completely new topic (e.g., a "Quantum Mechanics" paper when you only have "Finance" folders), it detects the anomaly, uses Generative AI to create a new category, and rebuilds its own sorting index instantly.

How It Works

The system follows a decision pipeline: Identify ‚Üí Search ‚Üí Generate (if needed) ‚Üí Move.

1Ô∏è‚É£ Semantic Classification

Files are read (via text or OCR), encoded into vectors, and compared against the folder_embeddings.faiss index. If similarity is high (>0.5), it moves to the existing folder.

2Ô∏è‚É£ Generative Fallback (The AI Layer)

If confidence is low, the file content is sent to Google Gemini via the API. The LLM:

  • Analyzes the text summary.
  • Generates a broad Category Label (e.g., "Medical Records").
  • Creates a dense list of description keywords.
  • Updates folder_labels.json and triggers a FAISS index rebuild.

3Ô∏è‚É£ Real-time Watcher

The system monitors directories for Created or Moved events, ignoring temporary download files (.crdownload) until they are fully written.

Installation & Setup

Requirements

Python 3.8+, Google Cloud API Key, and Tesseract OCR.

pip install sentence-transformers faiss-cpu numpy pdfplumber pytesseract pillow python-docx watchdog pystray google-genai python-dotenv

Environment Config

Create a .env file in the root:

API_KEY=your_gemini_api_key_here

Quick Start

  1. Clone: git clone https://github.com/ahhyoushh/filesense.git
  2. Init: python scripts/create_index.py (Creates initial index)
  3. Run GUI: python scripts/launcher.py
